PHP 序列化与反序列化基础

基本概念

序列化 (Serialization)

序列化是将内存中的对象转化为可存储/传输的字符串的过程。

PHP函数：serialize()

反序列化 (Deserialization)

反序列化是将字符串/字节流还原为内存中对象的过程。

PHP函数：unserialize()

应用场景

游戏存档

保存游戏进度，将角色属性、装备、位置等对象序列化成字符串存储到文件或数据库中。

数据传输

不同服务间传递对象，如Session存储、API调用、缓存等场景。

PHP 序列化格式详解

示例代码


class User {
    public $name = "Alice";
    public $age = 20;
    public $isAdmin = false;
}

$user = new User();
serialized = serialize($user);
echo $serialized;
输出结果 
O:4:"User":3:{s:4:"name";s:5:"Alice";s:3:"age";i:20;s:6:"isAdmin";b:0;}

序列化格式解析

格式 含义 示例
O:4 对象，类名长度为4 O:4:"User"
s:4:"name" 字符串，长度为4，值为"name" s:4:"name";s:5:"Alice"
i:20 整数，值为20 i:20
b:0 布尔值，false b:0 (true为b:1)
N NULL值 N;
a:3 数组，包含3个元素 a:3:{...}

反序列化示例

反序列化操作


// 接收到序列化的字符串
$serialized = 'O:4:"User":3:{s:4:"name";s:5:"Alice";s:3:"age";i:20;s:6:"isAdmin";b:0;}';

// 反序列化还原为对象
$user = unserialize($serialized);

// 访问对象的属性
echo $user->name;      // 输出: Alice
echo $user->age;       // 输出: 20
var_dump($user->isAdmin); // 输出: bool(false)
输出结果 
Alice
20
bool(false)

反序列化漏洞利用条件

同时满足以下三个条件才可利用反序列化漏洞

1

存在反序列化入口

$a = unserialize($_POST['data'])

输入可控的反序列化点

2

存在危险操作

魔术方法能触发危险函数

如 system(), exec(), file_put_contents()

3

可构造POP链

魔术方法能在反序列化时/后被触发

需构造完整的调用链

常见数据类型序列化格式

String s:5:"hello"

长度5的字符串"hello"

Integer i:114514

整数114514

Boolean b:1

布尔值true

Float d:3.14159

浮点数3.14159

Null N;

NULL值

Array a:2:{i:0;s:3:"key";i:1;s:5:"value";}

包含2个元素的数组
